为了提高东华大学思政考试的成绩，方便学生的做题，同时模拟出考试场景，特编写出思政刷题系统。本软件绿色无毒，完全免费，也无广告，放心使用，谁叫我也是东华的呢？嘿嘿。。。

* 支持马克思，毛概，思修，近代史做题等
